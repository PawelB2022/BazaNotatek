# BazaNotatek

Jest to projekt bazodanowy wykorzystujący bazę danych typu SQLite. Do jego obsługi użyto język Java, wykorzystując prz tym framework Hibernate do manipulacji bazą danych oraz JavaFX do graficznego interfejsu.
Założeniem programu jest prosta obsługa notatek. Użytkownik jest w stanie utworzyć profil z hasłem, gdzie ma dostęp do tworzenia, edycji i usuwania notatek utworzonych na swoim profilu.
Użytkownik może też zmienić swoją obecną nazwę oraz usunąć swój profil, co spowoduje usunięcie wszystkich notatek powiązanych z tym użytkownikiem.
Istnieje także prosta obsługa filtrowania notatek poprzez tytuł notatki oraz datę jej utworzenia i ostatniej modyfikacji.
